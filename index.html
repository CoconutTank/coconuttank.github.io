Die Parameters

Min die value:

Max die value:

Die parameters have the following expectations:

  • Die values must be actual integers.
  • The minimum die value cannot be greater than or equal to the maximum die value.
  • The minimum die value cannot be less than 1.
  • Each individual die parameter pair will be validated.

Die Pool

Die Pool Input:

The die pool input has the following expectations:

  • Each individual die parameter pair consists of two integers, concatenated together with commas(,) inbetween.
  • All die parameter pairs in a die pool input are concatenated together with semi-colons(;) inbetween.
  • Only integers, commas(,) and semi-colons(;) are allowed.
  • Any non-integer number will be truncated to its floor value.
  • Each individual die parameter pair will be validated.
Any die pool string which was previously generated by this dice simulator should be acceptable input.

Examples of valid die pool inputs:

  • 1,2
  • 1,3;1,4;1,5
  • 1,6;3,6;1,6;10,20
